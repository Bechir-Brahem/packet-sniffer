# Presentation

This is a simple packet sniffer written in c/c++ based on a tui and using raw sockets  
it also understands packet header information of various protocols ethernet,arp,ip,icmp  

![arp](images/arp.png)
![icmp](images/icmp.png)
![udp](images/udp.png)

# Setup
```bash
make
sudo ./sniffer
```
